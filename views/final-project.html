Introduction

For my final project this year, I was assigned to help out with constructing and coding the robot for this years VEX robotics competition. The robot's main goal is to be able to pick up, and drop blocks intoour team alliance's corner, to gain as many points as we can.

Engineering

For the robot, we decided to build it on a H-drive. With it being built on a H-drive, it allows for the robot to move the standard forwards and back, however there is another wheel in the middle allow for the robot to also shuffle left and right, allowing for greater mobility.

To pick up the blocks, we're using a intake tread device inorder to lift them. Once one block is picked up, a second can be picked up. The blocks will continue to stack ontop of eachother, allowing us to pick up 4-5 blocks. We then use a piston device to push a main platform up, then to allow us to drop the blocks off into a safe position stacked on eachother.

Strategy

Our main plan come competition time, is to collect as many cubes as we can, ignoring the towers, and moving straight back to our alliance area, dropping the blocks in a stack onto the area. The towers in our opinion would be a waste of time, due to the fact that it would take time and allignment effort to place a single block into the tower. It's better to collect as many blocks as we can to place and gain as much points as we can before time runs out.
